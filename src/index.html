Einfach und schnell Zitationen suchen

VizArt ist ein Tool, welches Forschern das Betrachten von Zitier-Verbindungen in Fachliteratur erleichtert. Der Name steht für die Visualisierung von Artikeln.

Forscher nutzen diesen Service, um eine Artikel-Menge für eine Untersuchung aufzubereiten - beispielsweise Untersuchung ihres Abdeckungsgrads durch Reviews. Dafür werden Publikationsdaten aus PubMed und Zitationsmerkmale aus iCite extrahiert und in einem geeigneten Graphnetzwerk-Format zusammengeführt. Exportiert, kann die Datei z.B. in Gephi einer SNA (Social Network Analysis) unterzogen werden.
